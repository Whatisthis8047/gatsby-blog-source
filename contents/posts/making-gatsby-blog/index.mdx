# Gatsby 튜토리얼

해당 글은 제가 gatsby 문서([getting-started](https://www.gatsbyjs.com/docs/tutorial/getting-started/part-0/))를 바탕으로 진행한 내용입니다.\
보다 전문적이고, 확실한 내용은 해당 링크를 참조하세요!

`Gatsby` 프레임워크를 이용하기 위해서는 몇가지 사전 지식이 필요합니다. 먼저 Html, CSS 그리고 Java Script 언어입니다.\
(네, 전 편을 보셨다면 아시겠지만 저는 지금 블로그 하나 호스팅한다고 `ruby`와 `go`를 지나 이거까지 무려 3개의 언어랑 씨름하는 중입니다.)
\
물론 셋 모두에 전문가일 필요는 없습니다(그리고 전문가시면 이 글을 안보셔도... 될걸요?) 저도 전문가는 아니지만 그때그때 필요한 내용을 찾아가며 만들고 있으니 걱정하지 말고 일단 시작하셔도 좋습니다!
